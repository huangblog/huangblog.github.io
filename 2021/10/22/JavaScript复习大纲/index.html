三、简答题：

1、什么是 DOM？

文档对象模型（Document Object Model）、通过 DOM 可以动态改变操控文档（网页）内容。例如改变网页背景色、table、tr 颜色、字体样式、输入框的值等一切网页内容。

2、获取 DOM 元素的方法有哪些？

通过 id 选取元素：getElementById()

通过 name 属性值选取元素集合： getElementsByName()

通过 tag 标签名选取元素集合：getElementsByTagName()

通过 css 类选取元素集合：getElementsByClassName()

3、js 中常用的数据类型有哪些？并说明作用

1、number 数字

2、boolean 布尔值

3、undefined 未定义的值

4、null 空值

5、string 字符串

4、js 中常用的事件有哪些？并说明作用

1、onblur()：失去焦点时触发

2、onchange()：选项发生改变时触发

3、onclick()：在对象被点击时调用事件

4、onmouseover()：鼠标移到某元素之上时会触发事件

5、onmouseout()：鼠标从某元素移开会触发事件

6、onmousedown()：鼠标按钮被按下会触发事件

5、Document 对象方法有哪些？

open()

close()

write()

writeln()
